# 向量和矩阵

3 行 2 列的矩阵

列向量

3 行一列的向量 从 1 开始，步长为 0.1，一直增长到 2

v = 1:0.1:2

另外一些生成矩阵的方法

C = ones(2, 3)

ones()函数来生成一个矩阵，上面的的例子生成了一个 2 行 3 列的矩阵，矩阵里的值都为 1。我们还可以这样

C = 3*ones(2, 3)

C =
  3   3   3
  3   3   3

等同于下面这样的操作

C = [3 3 3; 3 3 3]

v = once(1, 3)

x = zeros(1, 3)

x = rand(1, 3)

zeros() 生成 1 行 3 列的值为 0 的矩阵 rand() 生成 1 行 3 列的值为 0~1 的随机值的的矩阵

高斯平局分布

w = randn(1, 3)

randn()生成了 1 行 3 列的 3 个值，它们服从高斯分布，均值为 0。标准差或者方差为 1

w = -6 + sqrt(10) * randn(1, 10000)
hist(w)
hist(w, 20)

w 是一个以根号 10 乘以高斯随机变量生成的矩阵, 均值为-6 的高斯分布直方图，这个高斯随机变量的方差为 10， 其标准差是根号 10，约等于 3.1 我们可以用hist(w) 函数来生成w直方图

eye()函数表示生成单位矩阵

H = eye(4)

A = [1, 2; 3, 4; 5, 6]

   4   5

R = A * C

我们将矩阵 A 与矩阵 C 相乘，得到下面的结果

R =
   10   13
   22   29
   34   45

具体的计算过程为： A 的第一行与矩阵 C 相乘

1 * 2 + 2 * 4 = 10
1 * 3 + 2 * 5 = 13

A 的第二行与矩阵 C 相乘

3 * 2 + 4 * 4 = 22
3 * 3 + 4 * 5 = 29

A 的第三行与矩阵 C 相乘

5 * 2 + 6 * 4 = 34
5 * 3 + 6 * 5 = 45

如果我们将 A * B 会得到一行错误

error: operator *: nonconformant arguments (op1 is 3x2, op2 is 3x2)

错误的原因是行列的个数不对，A 的第一行只有 2 个元素，而 B 的第一列有 3 个元素，所为无法相乘，A 与 B 的计算可以是这样

A .* B

ans =
   10   22
   36   52
   70   90

结算结果是用 A 与 B 的对于元素相乘

1 * 10 = 10
2 * 11 = 22

3 * 12 = 36
4 * 13 = 52

5 * 14 = 60
6 * 15 = 90

# 数据绘制

t = [0:0.01:0.98];
y1 = sin(2 * pi * 4 * t)
plot(t, y1)

y2 = cos(2 * pi * 4 * t)
plot(t, y2)

t 是一个从0开始，每次递增0.01，直到0.98的向量 y1 和 y2 是在t向量的基础上乘上pi生成的数据

plot(Y) 函数将传递的数据生成图像

 -- plot (Y)
 -- plot (X, Y)
 -- plot (X, Y, FMT)
 -- plot (..., PROPERTY, VALUE, ...)
 -- plot (X1, Y1, ..., XN, YN)
 -- plot (HAX, ...)

hold on 保持当前所绘制的图像

plot(t, y1);

hold on;

plot(t, y2, 'r')

xlabel(str) 显示在x轴上的内容

xlabel('time')

ylabel(str) 显示在y轴的内容

ylabel('value')

legend(str1, str2, str3, ..) 在右上角显示sin 和 cos曲线各自的颜色

legend('sin', 'cos')

title(str) 显示title

title('my plot')

figure(number) 可以同时显示多张图表

figure(1); plot(t, y1)
figure(2); plot(t, y2)

subplot(number, number, number) 前2个参数是将图像分为1*2个窗格，后一个参数是将数据生成在指定的窗格上

subplot(1, 2, 1);

将图片分为高度为100%的2个窗格，将数据写入第一个窗格

axis(Array) 第一个参数为数组参数，参数前2个值分别是x轴的x1，x2，关系是从低到高的数值范围，后2个参数是y1， y2轴，第5和6个参数是z轴的z1, z2 -- axis () -- axis ([X_LO X_HI]) -- axis ([X_LO X_HI Y_LO Y_HI]) -- axis ([X_LO X_HI Y_LO Y_HI Z_LO Z_HI]) -- axis ([X_LO X_HI Y_LO Y_HI Z_LO Z_HI C_LO C_HI]) -- axis (OPTION) -- axis (OPTION1, OPTION2, ...) -- axis (HAX, ...)

axis([0.5, 1, -1, 1])

使用,逗号可以同时输入多个命令

A = magic(5)

imagesc(A)
imagesc(A), colorbar, colormap gray;

首先是使用magic(5)生成一个5*5的上下左右及对角线每个数字之和都相等的矩阵，然后通过colorbar来显示颜色条，colormap gray来显示颜色
